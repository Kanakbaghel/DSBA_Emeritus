# Curated Resources on Machine Learning and Data Mining

This resources curates a collection of academic papers, surveys, and books on key machine learning topics, including feature selection, logistic regression, decision trees, hyper-parameter optimization, time series forecasting, ensemble learning, and data mining techniques. These resources provide theoretical foundations, practical applications, and advanced insights for researchers, students, and practitioners in data science and AI.

## Resources Overview

### Feature Selection
| Title | Resource Link | Description |
|-------|---------------|-------------|
| **Feature Selection: A Data Perspective – Jundong Li et al. (2016)** | [arXiv Paper](https://arxiv.org/abs/1601.07996?utm_source=chatgpt.com) | A comprehensive survey that organizes feature selection approaches into four main methodological categories—similarity-based, information-theoretic, sparse learning, and statistical methods—while exploring their application across diverse contexts, including conventional, structured, heterogeneous, and streaming data. |

### Hyper-Parameter Optimization
| Title | Resource Link | Description |
|-------|---------------|-------------|
| **Hyper-Parameter Optimization: A Review of Algorithms and Applications** | [arXiv Paper](https://arxiv.org/abs/2003.05689?utm_source=chatgpt.com) | Tong Yu & Hong Zhu (2020) present a comprehensive overview of hyperparameter optimization. ### Time Series Forecasting
| Title | Resource Link | Description |
|-------|---------------|-------------|
| **Machine Learning Advances for Time Series Forecasting** | [arXiv Paper](https://arxiv.org/abs/2012.12802?utm_source=chatgpt.com) | “Machine Learning Advances for Time Series Forecasting” by Masini et al. (2020) — Discusses both linear methods (penalized regression, ensembles) and nonlinear approaches (neural networks, tree-based models), including hybrid techniques, predictive performance, and practical real-world applications. |
| **Time Series Analysis and Modeling to Forecast: A Survey** | [arXiv Paper](https://arxiv.org/abs/2104.00164?utm_source=chatgpt.com) | “Time Series Analysis and Modeling to Forecast: A Survey” by Dama & Sinoquet (2021) — Offers a unified overview from preprocessing to forecasting, covering linear models, nonlinear extensions, deep learning categories, and widely used open-source software tools. |
